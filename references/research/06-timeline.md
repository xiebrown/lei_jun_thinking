# 雷军人物时间线

## 早年（1969-1991）

| 时间 | 事件 | 思想意义 |
|------|------|---------|
| 1969.12.16 | 出生于湖北仙桃 | 普通家庭出身 |
| 1987 | 考入武汉大学计算机系 | 命运转折点 |
| 1987-1989 | 两年修完四年学分 | "勤奋可以弥补差距"的早期信念 |
| 1989 | 读《硅谷之火》，受乔布斯启发 | **第一次思想觉醒**："我也想创办一家伟大的公司" |
| 1989-1991 | 与同学尝试第一次创业（仿制金山汉卡） | 早期创业经验，以失败告终 |
| 1991 | 武汉大学毕业 | 正式进入社会 |

## 金山时期（1992-2007）

| 时间 | 事件 | 思想意义 |
|------|------|---------|
| 1992 | 加入金山软件，第6号员工 | 正式进入软件行业 |
| 1998 | 29岁被推举为金山CEO | **"赶鸭子上架"式转型**，从程序员到管理者 |
| 1999 | 金山第一次尝试IPO | 开始"熬"的漫长上市路 |
| 2000 | 创办卓越网（电商） | 早期互联网探索 |
| 2004 | 出售卓越网给亚马逊（7500万美元） | 获得第一桶金，开始天使投资 |
| 2004 | 第一笔天使投资：拉卡拉 | 开启投资人身份 |
| 2007 | **金山第五次IPO成功，港交所上市** | "熬"到终点，旋即辞职 |
| 2007.12 | 辞去金山CEO职务 | **"顺势而为"认知形成**：反思金山错过了互联网大潮 |

## 天使投资时期（2007-2010）

| 时间 | 事件 | 思想意义 |
|------|------|---------|
| 2007-2010 | 作为天使投资人投资YY、UC、拉卡拉等 | 形成"只投人不投项目"的投资哲学 |
| 2008 | 40岁前实现财务自由 | **"中年危机"**：有钱但没有新目标 |
| 2009 | 深度反思"为什么金山那么努力却没有大成" | 形成"顺势而为"的核心认知 |
| 2010.4 | 与林斌等联合创立小米 | **"40岁清零重新开始"** |

## 小米爆发期（2010-2015）

| 时间 | 事件 | 思想意义 |
|------|------|---------|
| 2010.8 | MIUI发布 | 软件先行策略 |
| 2011.8 | 小米手机1发布，定价1999元 | **"互联网思维做手机"**引爆市场 |
| 2012 | 销售额126亿 | 爆发式增长 |
| 2013 | 与董明珠10亿赌局 | 过度自信的巅峰时刻 |
| 2014 | **中国手机市场第一** | 创业4年登顶 |
| 2015 | "Are You OK"鬼畜视频走红 | 意外成为"网红"企业家 |

## 小米低谷与反弹（2015-2019）

| 时间 | 事件 | 思想意义 |
|------|------|---------|
| 2015-2016 | 供应链危机、销量下滑36%、三星断供 | **最严重的经营危机** |
| 2016.5 | **雷军亲自接管手机部** | 创始人回归，"豁出去" |
| 2016 | 4次飞韩国三星总部修复供应链 | 放下身段解决问题 |
| 2017 | 小米印度市场第一、营收破1000亿 | 海外突破带来反弹 |
| 2018.7 | 小米港交所上市，当天破发 | 第一次资本市场挫败 |
| 2018 | 董事会决议：硬件净利润率永不超过5% | **制度化"性价比"承诺** |
| 2019 | 推出Redmi品牌，小米品牌向上 | 双品牌战略启动 |

## 小米高端化与造车（2020-2025）

| 时间 | 事件 | 思想意义 |
|------|------|---------|
| 2020.8 | 小米十周年，首次年度演讲 | 开始系统化输出思想 |

| 2021.3 | 雷军宣布"押上一生积累的战绩和声誉，为小米汽车而战" | **"人生最后一次创业"** |
| 2022 | 《小米创业思考》出版 | 方法论系统化输出 |

| 2024.3.28 | 小米SU7发布 | **造车首战告捷**，4分钟大定破万 |

| 2025.10 | 成都SU7事故、116887辆召回 | 安全问题持续发酵 |

## 思想转折点标注

| 时间 | 转折 | 影响 |
|------|------|------|
| 1989 | 读《硅谷之火》 | 确立"创办伟大公司"的梦想 |
| 2007 | 离开金山 | 形成"顺势而为"核心认知 |
| 2010 | 40岁创立小米 | 从投资人回归创业者，"豁出去" |
| 2016 | 亲自接管手机部 | 从"导师型CEO"到"战斗型CEO" |

| 2023 | "99%的问题有标准答案" | 方法论从感性到理性的成熟 |
| 2025 | 安全事故后的反思 | 从"流量神话"到"责任担当"的被迫转型 |
